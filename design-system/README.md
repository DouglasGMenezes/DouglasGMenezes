# Design System

Este diretório define a identidade visual do repositório.

## Paleta

- `--bg`: `#0b1020`
- `--surface`: `#111827`
- `--surface-2`: `#162033`
- `--border`: `#263244`
- `--text`: `#e5eef8`
- `--muted`: `#93a4ba`
- `--accent`: `#7dd3fc`
- `--accent-strong`: `#38bdf8`

## Tipografia

- Títulos: `Inter`, `Segoe UI`, `system-ui`
- Corpo: `Inter`, `Segoe UI`, `system-ui`
- Código e labels técnicas: `SFMono-Regular`, `JetBrains Mono`, `ui-monospace`

## Espaçamento

- Base: `4px`
- Tokens: `4`, `8`, `12`, `16`, `24`, `32`, `48`, `64`
- Os layouts devem priorizar respiro e alinhamento óptico.

## Cartões

- Fundo escuro com borda sutil.
- Raio moderado.
- Sombra mínima.
- Títulos curtos e descrição objetiva.

## SVGs

- Traço fino.
- Poucas cores.
- Grid discreto.
- Gradientes suaves, sem ruído visual.
- Uso de `defs` para manter consistência.

## Componentes Reutilizáveis

- Banner principal
- Divider de seção
- Card de arquitetura
- Logo do projeto Fintech Microservices
